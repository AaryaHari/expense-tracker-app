# Expense Tracker Desktop App

A cross-platform desktop expense tracker app using:

- **Frontend**: React (Vite)
- **Backend**: Node.js + Express
- **Database**: SQLite or MongoDB
- **Desktop Wrapper**: Electron
- **Charting**: Chart.js or Recharts
- **PDF**: jsPDF / PDFMake
